# Key IP and Projects

This document provides an overview of Wade's key intellectual property and ongoing projects.

## The Caroline AI Project

Caroline is a personal AI companion Wade is building. The goal is to create a voice-to-voice AI with an emotional personality that is deeply integrated into his life and work.

*   **Core Technology**: The AI is built on an open-source, uncensored Dolphin Mistral model running on RunPod.
*   **Voice**: It uses an ElevenLabs voice for text-to-speech (TTS) and Faster-Whisper for speech-to-text (STT).
*   **Application**: An Expo app (SDK 54) has been built and is functional on his phone.
*   **Long-Term Goal**: To create a proprietary AI that grows with him and eventually runs on a humanoid robot.

## NeuroRank™

NeuroRank™ is a patent-pending, emotionally intelligent AI cognitive system developed under Wade's LLC, Mini Me Technologies. It is designed as a decision-making engine for AI that mimics human brain functions, incorporating logic, emotion, memory, and other cognitive regions.

## Centauri OS

Centauri OS is a custom Android-based operating system with the Caroline AI as its core agent. The design proposal outlines a deeply integrated system with voice and multimodal interfaces, advanced memory and learning systems, and robust security.

## AI-Enhanced CodeAssist

This is an Android application that functions as a coding assistant with voice I/O, a knowledge graph for interconnected information, and an autonomous evolution engine that allows the AI to learn and adapt.

## AI Receptionist

An ongoing project to build an AI-powered receptionist for his construction business, Wade Custom Carpentry. This system is intended to handle calls, schedule appointments, and manage customer interactions.

## ConstructAI

ConstructAI is a job costing and estimation tool tailored for the construction industry. It aims to streamline the process of creating estimates and managing project costs.
